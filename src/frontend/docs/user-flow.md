# KindLink User Flow

## First-Time User Journey

### 1. Landing & Authentication
1. User arrives at KindLink landing page
2. Sees welcome message and "Get Started" button
3. Clicks button to initiate Internet Identity login
4. Completes Internet Identity authentication
5. Returns to KindLink as authenticated user

### 2. Onboarding
1. **Name Entry**
   - Prompted to enter display name
   - Name will be visible to community (unless posting anonymously)
   - Clicks "Continue" to proceed

2. **Community Selection**
   - Presented with list of sample communities
   - Can search communities or enter custom community name
   - Selects their school or neighborhood
   - Profile saved with name and community

### 3. Main App Access
- Automatically navigated to Good Deeds feed
- Bottom navigation visible with 5 sections:
  - Chat
  - Help
  - Challenge
  - Good Deeds
  - Settings

## Core User Flows

### Chat Rooms Flow
1. Tap "Chat" in bottom navigation
2. View list of community chat rooms
3. **Create New Room:**
   - Tap "New Room" button
   - Enter room name
   - Room created and user navigated to it
4. **Join Existing Room:**
   - Tap on room card
   - View messages and community guidelines
   - Type message and send
   - Report or mute/block users via actions menu

### Help Requests Flow
1. Tap "Help" in bottom navigation
2. View list of help requests in community
3. **Post Help Request:**
   - Tap "Ask for Help" button
   - Select category
   - Write description
   - Toggle anonymous posting (default: on)
   - Submit request
4. **View Help Request:**
   - Tap on request card
   - Read full description
   - Report if needed via actions menu

### Daily Challenge Flow
1. Tap "Challenge" in bottom navigation
2. View today's kindness challenge
3. **Complete Challenge:**
   - Read challenge description
   - Optionally write private reflection
   - Check "I completed this challenge"
   - Completion saved locally
4. **Share Completion (Optional):**
   - After marking complete, tap "Share Anonymously"
   - Write about experience
   - Posted as anonymous good deed with challenge tag

### Good Deeds Flow
1. Tap "Good Deeds" in bottom navigation (or default landing)
2. View chronological feed of good deeds
3. **Share Good Deed:**
   - Tap "Share a Good Deed" button
   - Write about kind act
   - Add optional tags (up to 5)
   - Toggle anonymous posting (default: on)
   - Submit
4. **View Good Deeds:**
   - Scroll through feed
   - Tap refresh to load new posts
   - Report or mute/block via actions menu

### Settings & Community Management Flow
1. Tap "Settings" in bottom navigation
2. **Edit Profile:**
   - Update display name
   - Save changes
3. **Change Community:**
   - Tap "Change Community"
   - Select new community from picker
   - All content updates to new community
4. **Moderator Access (Admins Only):**
   - Tap "Moderator" button
   - Access moderator dashboard
5. **Logout:**
   - Tap "Log Out"
   - Confirm logout
   - All cached data cleared
   - Returned to landing page

### Moderation Flow (Admins Only)
1. Access moderator dashboard from settings
2. View table of content reports
3. Filter by status (All, Open, Under Review, Resolved)
4. **Review Report:**
   - Click "Review" on report row
   - View report details (type, reason, timestamp, content ID)
   - Mark as "Under Review" or "Resolved"
   - Status updated in system

## Safety & Reporting Flow
1. User encounters concerning content
2. Taps report icon (alert triangle) on content
3. Report dialog opens
4. User writes reason for report
5. Submits report
6. Confirmation message shown
7. Report appears in moderator dashboard
8. Moderator reviews and takes action

## Returning User Journey
1. User returns to KindLink
2. Automatically authenticated if session valid
3. Profile and community loaded
4. Lands on Good Deeds feed
5. Continues using app from any section
