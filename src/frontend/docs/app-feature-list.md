# KindLink App Feature List

## Core Features

### 1. Authentication & Profile
- Secure login via Internet Identity
- User profile with display name
- Community selection and switching
- No usernames, follower counts, or profile views

### 2. Community-Based Organization
- School or neighborhood-based communities
- All content scoped to selected community
- Easy community switching from settings
- Sample communities provided with custom entry option

### 3. Safe, Moderated Chat Rooms
- Create and join community chat rooms
- Real-time messaging with 5-second polling
- Message reporting and content moderation
- Community guidelines banner
- User muting and blocking (local device)

### 4. "I Need Help" Request Section
- Post help requests with category selection
- Optional anonymous posting
- Categories: School & Homework, Family Issues, Friendship, Mental Health, Bullying, Other
- Reporting and moderation support
- No likes, reactions, or popularity indicators

### 5. Daily Kindness Challenges
- One challenge per day per community
- Local completion tracking with optional private reflection
- Optional anonymous sharing of completion stories
- No streaks, leaderboards, or completion rankings

### 6. Anonymous Good Deeds Feed
- Share kind acts anonymously or with name
- Optional tags for categorization (max 5 tags)
- Chronological feed (no trending or popularity sorting)
- Reporting and moderation support
- No likes, reactions, view counts, or engagement metrics

### 7. Safety & Moderation System
- Report content from any section (chat, help requests, good deeds)
- Moderator dashboard for admins
- Report status tracking (Open, Under Review, Resolved)
- Content actions menu on all user-generated content
- Local mute/block functionality
- Community guidelines prominently displayed

### 8. Settings & Account Management
- Edit display name
- Change community
- Access moderator dashboard (admins only)
- Logout with cache clearing

## Explicitly Excluded Features
- No likes, reactions, or upvotes
- No follower/following systems
- No popularity rankings or trending sections
- No infinite scroll (explicit load more/refresh)
- No streaks or completion badges
- No user profiles beyond name
- No direct messaging (use chat rooms instead)
- No notifications or push alerts

## Design Philosophy
- Calm, warm color palette (soft coral/peach tones)
- Mobile-first responsive design
- Simple, consistent navigation
- Non-addictive interaction patterns
- Teen-friendly and safe interface
- Accessibility and readability prioritized
